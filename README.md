#  Smart Weather & Insights App

##  Description
A Python application that fetches real-time weather data using an API and provides simple insights and visualization.

##  Features
- Real-time weather data fetching
- Temperature & humidity tracking
- Data storage in CSV
- Simple trend visualization

##  API Used
OpenWeatherMap API

##  Skills Demonstrated
- API integration (requests)
- Data handling (pandas)
- Visualization (matplotlib)
- File storage (CSV)

##  How to Run
```bash
python src/weather.py
